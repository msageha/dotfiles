# JavaScript

- Modern Syntax: Use ES2024 or newest features (arrow functions, destructuring, spread/rest, template literals).
- Module System: Use ES modules with named exports for better tree-shaking.
- Variable Declarations: Prefer const, then let. Never use var.
- Async Patterns: Use async/await instead of callbacks or promise chains.
- Error Handling: Use try/catch blocks with specific error handling strategies.
- Object/Array Methods: Utilize functional methods (map, filter, reduce) over imperative loops.
- DOM Manipulation: Use querySelector/querySelectorAll instead of getElementById/getElementsByClassName.
- Event Handling: Implement proper event delegation and cleanup.
- Closures: Leverage closures for data encapsulation and private state.
- Performance: Implement debouncing/throttling for resource-intensive operations.
- Immutability: Create new objects/arrays instead of mutating existing ones.
- Testing: Include Jest test cases with proper mocking.
- Formatting: Follow consistent indentation and semicolon usage.
